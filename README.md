# 📚 Projetos em Python

Este repositório contém uma coleção de **projetos simples desenvolvidos em Python**, criados com o objetivo de praticar **lógica de programação**, **estruturas básicas da linguagem** e **resolução de problemas**.

## 🚀 Projetos

### 🎮 Pedra, Papel e Tesoura

Implementação do clássico jogo **Pedra, Papel e Tesoura**, onde o usuário joga contra o computador.
O programa gera uma escolha aleatória para o computador e determina o vencedor com base nas regras do jogo.

**Conceitos aplicados:**

* Estruturas condicionais (`if`, `elif`, `else`)
* Geração de números aleatórios
* Entrada de dados do usuário

---

### ❌⭕ Jogo da Velha

Versão do tradicional **Jogo da Velha** executado no terminal.
Permite que dois jogadores se alternem nas jogadas até que haja um vencedor ou empate.

**Conceitos aplicados:**

* Listas e matrizes
* Estruturas de repetição
* Validação de jogadas
* Lógica para verificação de vitória

---

### 🧮 Calculadora em Python

Uma **calculadora simples** capaz de realizar operações matemáticas básicas diretamente pelo terminal.

**Operações disponíveis:**

* Adição
* Subtração
* Multiplicação
* Divisão

**Conceitos aplicados:**

* Funções
* Estruturas condicionais
* Entrada e saída de dados

---

## ▶️ Como Executar os Projetos

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse a pasta do projeto:

```
cd seu-repositorio
```

3. Execute o arquivo Python desejado:

```
python nome_do_arquivo.py
```
